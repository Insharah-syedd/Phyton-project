# Phyton-project
A repository of mini Python projects, scripts, and practice exercises for learning and experimenting.

Welcome! This repository contains a collection of small Python projects that run entirely in the terminal. These projects are perfect for learning Python and practicing coding skills.

 Projects

Here are the projects in this repository:

- **calculator.py** – A simple command-line calculator for basic math operations.
- **chatbot.py** – A basic chatbot that can chat with the user in the terminal.
- **currencyconverter.py** – Convert amounts between different currencies in the terminal.
- **epicadventurequiz.py** – An adventure-themed quiz game played in the terminal.
- **first.py** – A beginner Python script to practice coding.
- **quiz.py** – A terminal-based quiz game to test your knowledge.
- **resturant.py** – A simple restaurant order management system in the terminal.
- **snake.py** – Classic Snake game implemented to run in the terminal.
- **studentresultmangement.py** – Manage and calculate student results (grades/marks) using terminal input.
- **tictac.py** – Tic-Tac-Toe game for two players in the terminal.
